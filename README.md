# Lankasa

A personal, modular Hyprland desktop environment built from scratch.

## Goals

- Modular architecture
- Maintainable configuration
- Theme engine
- Elegant desktop experience

## Project Structure

```text
engine/     Core engine
themes/     Theme definitions
scripts/    Startup utilities
config/     Shared configuration
bin/        CLI (coming soon)
```

> This project is designed for personal use while keeping production-quality engineering practices.
