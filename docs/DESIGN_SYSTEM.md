# Lankasa Design System

## Philosophy

Every theme in Lankasa is more than a color palette.

A theme defines:

- Mood
- Visual language
- Component behavior
- Material
- Animation

## Component Rule

Every component should follow the active theme identity.

Components:

- Waybar
- Kitty
- Wofi
- Hyprlock
- SwayNC

No component should define its own visual identity outside the active theme.
